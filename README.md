# 🎥 Multi-Camera Object Tracking & Heatmap System

## 📌 Project Overview

This project implements a real-time multi-camera object tracking system integrated with movement heatmap visualization.

The system connects three independent camera streams, performs object detection and tracking, and generates a dynamic heatmap to analyze movement density.

Each component is modular and separated for scalability and maintainability.

---

## 🚀 Key Features

- 🎥 Three independent camera connections
- 🧠 Real-time object detection
- 🔄 Object tracking with unique IDs
- 🔥 Dynamic heatmap generation
- 🏗 Modular architecture (separate components)
- 📊 Movement density analysis
- ⚡ Designed for scalability

---

## 🧠 Tech Stack

- Python
- OpenCV
- YOLO (Object Detection)
- SORT / DeepSORT (Object Tracking)
- NumPy
- Matplotlib
